# bmp

Parse and build BMP files in crystal lang.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     bmp:
       github: Globoplox/bmp
   ```

2. Run `shards install`

## Usage

```crystal
require "bmp"

```
